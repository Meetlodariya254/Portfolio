## 📖 Overview

A sleek, high-performance personal portfolio website built for **Meet Lodariya** (Full Stack Web Developer). Developed using **React 19, Vite, and modular Vanilla CSS**, it features modern glassmorphic aesthetics, custom mouse-following glow effects, smooth scroll-reveal micro-animations, interactive 3D project cards, and an integrated EmailJS contact form. Designed from the ground up to showcase technical skills, featured projects, and professional experience.

---

## ✨ Key Features

* 🎨 **Rich Modern Aesthetics**: Designed with sleek glassmorphism, curated dark mode palettes, and subtle grid overlays.
* ⚡ **Dynamic Micro-Animations**:
  * **Typing Text Animation**: Custom typewriter effect cycling through core developer specialties.
  * **Interactive Mouse Glow**: Fluid cursor-tracking spotlight illuminating the Hero section.
  * **3D Card Tilt Effect**: Smooth cursor-responsive perspective tilt on project showcases.
  * **Staggered Scroll Reveals**: Performance-optimized intersection observers triggering smooth viewport entry transitions.
* 📬 **Integrated Contact System**: Direct client-side dispatch using `@emailjs/browser` with automatic validation and a 1-click fallback to Gmail compose.
* 📱 **100% Responsive Design**: Fluid typography and flexbox/grid architecture ensuring a seamless experience across phones, tablets, and ultrawide monitors.

## ⚙️ Local Development & Setup

Follow these steps to run the project locally on your machine:

### 1. Clone the repository
```bash
git clone https://github.com/Meetlodariya254/Portfolio.git
cd Portfolio
```

### 2. Install dependencies
```bash
npm install
```

### 3. Configure Environment Variables
Create a `.env` file in the root directory and add your EmailJS API credentials:

```env
VITE_EMAILJS_SERVICE_ID=your_service_id
VITE_EMAILJS_TEMPLATE_ID=your_template_id
VITE_EMAILJS_PUBLIC_KEY=your_public_key
```

### 4. Start the development server
```bash
npm run dev
```
Open [http://localhost:5173](http://localhost:5173) in your browser to view the application.
